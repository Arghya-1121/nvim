🌙 My Neovim Config (LazyVim + Flutter)

Welcome to my Neovim configuration, built on top of the LazyVim starter template — customized for an awesome Flutter & Dart development experience! 🚀
✨ Features

    ⚡ Built on LazyVim

    🐦 Flutter tools preconfigured with:

        Real-time widget guides

        Closing tag highlights

        Dev logs in split terminal

        flutter pub get on pubspec.yaml save

        Debugging via nvim-dap

        Telescope integration

    💡 Java LSP support via jdtls

    💬 Auto-formatting + LSP autocompletion

    🔭 Telescope + Treesitter support

    🖤 Minimal UI with rounded borders, dark theme-friendly

    🧠 Lazy-loading everything for speed

🧩 Plugins & Tools

Some of the key plugins I use:

    nvim-flutter-tools

    nvim-cmp for completion

    nvim-dap for debugging

    mason.nvim for LSP/DAP installer

    telescope.nvim for fuzzy finding

    jdtls for Java support

    lualine.nvim for a clean statusline

🧪 Debugging Support

    DAP is enabled for Flutter apps.

    Toggle dev logs manually instead of auto-open.

    Java debugging can be enabled with additional config if needed.

📦 Setup Instructions

git clone https://github.com/Arghya-1121/nvim.git ~/.config/nvim
nvim

    Make sure to install the following tools manually (or via Mason):

        flutter + dart

        jdtls (for Java LSP)

        eslint, prettier, etc. (if using for web too)
